# SchoolRoute — Bus Routes & Carpooling App for Parents and Students

**Platform:** iOS + Responsive Web &nbsp;|&nbsp; **Tools:** Figma + HTML/CSS/JS &nbsp;|&nbsp; **Duration:** Jan–Mar 2026

---

**[▶ View Live Prototype](https://yasir164.github.io/SchoolRoute-ux-case-study/schoolroute.html)**

## Overview

SchoolRoute is a mobile app, responsive website, and fully coded interactive prototype helping parents and students access school bus routes and schedules in one place. It enables trusted carpooling groups within school communities with real-time notifications for delays and changes.

AI tools were integrated throughout. A moderated usability study with 5 participants drove two critical P0 design changes.

---

## Usability Testing Results

| Priority | Issue | Fix |
|---|---|---|
| P0 | Safety concerns blocked carpool flow (3/5 parents) | Verified parent badge + privacy tooltip |
| P0 | Carpool not discoverable (2/3 parents) | Carpool shortcut card on home dashboard |
| P1 | Stop list readability too low | Larger font; auto-highlight nearest stop |
| P1 | Form input fields lacked guidance | Placeholder text + time-picker widget |
| Strength | Clean minimal dark design praised | Preserved as visual direction benchmark |

---

## Coded Prototype Features

- Animated GPS map with pulsing dots and SVG route lines
- Bus stop timeline with live 'arriving now' highlight
- Group chat with real message sending in group details
- Dynamic invite list from email input in create group
- Day selector toggle for carpool schedule
- Severity-coded alert feed (green / yellow / red)
- Toggle switches for all notification preferences
- Sign up / sign in tab switcher with name persistence
